# Dashboard Excel

## Entendendo o Desafio da DIO

O objetivo deste desafio é criar um **dashboard**, com foco na organização e visualização de dados. 
O objetivo é transformar dados brutos em informações visuais claras e úteis, permitindo uma análise eficaz do desempenho de vendas e a tomada de decisões baseadas em dados.

## O que Fiz

Criei um **dashboard interativo no Excel** (.xlsx) que inclui:
- **Gráfico principal**: Quantidade de assinaturas renováveis.
- **KPIs chave**: Total de vendas e total de clientes.
- **Filtros dinâmicos**: Anual / Mensal / Trimestral (atualizam todos os campos e o gráfico automaticamente).
- **Tabelas dinâmicas**: Resumo de dados com fórmulas para cálculos automáticos.
- **Layout profissional**: Cores consistentes, fontes claras e organização (Dados Brutos | Dashboard | Instruções).

O dashboard destaca **simplicidade, foco e relevância**, priorizando visuais para evitar sobrecarga.

## Dados Utilizados

- **Fonte**: Dados fornecidos pelo desafio DIO.
  - Período: Todo o ano de 2024.
  
- **Transformações**: Limpeza de dados, pivot tables e fórmulas para agregação.

## Como Reproduzir e Usar

1. **Baixe o arquivo**: Clique em `desafio.xlsx` e salve localmente.
2. **Abra no Excel** (versão 2016+ ou Microsoft 365; compatível com Google Sheets).
3. **Ative macros** (se necessário, para slicers interativos).
4. **Use os filtros**: Clique nos slicers para filtrar por ano/mensal/trimestral – gráfico e dados atualizam em tempo real.
5. **Visualize**: A aba "Dashboard" mostra tudo pronto para análise.

## Tecnologias e Boas Práticas Aplicadas

- **Excel Avançado**: Tabelas dinâmicas, slicers, gráficos condicionais, formatação condicional.
- **Princípios de Dashboards**: Foco em KPIs relevantes, visualizações equilibradas, fluxo lógico (visão geral → detalhes).
- **Dicas para Melhoria**: Integre Power Query para dados reais ou Power BI para versão web.

Este projeto demonstra habilidades em **análise de dados, visualização e Excel avançado**.
